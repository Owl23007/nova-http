# create-nova-http

Initializer package for Nova HTTP projects.

## Usage

```bash
npm create nova-http@latest my-app
```

Or:

```bash
npx create-nova-http@latest my-app
```

## Options

```bash
npm create nova-http@latest my-app -- --template api --lang ts
```

Available options:

- --template minimal|api
- --lang ts|js
- --force
